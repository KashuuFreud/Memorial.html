# Memorial.html
upload some of my homework
作为初学者搭建一个普通的网页，仅供参考；
未插入图片视频、文字部分均为随机生成，仅供娱乐；
结构如下：
ses-memorial/
├── index.html          # 首页
├── gallery.html       # 相册页
├── quotes.html        # 语录页
├── timeline.html      # 时间轴页
├── contact.html       # 留言板页（包含预览返回修改功能）
├── css/
│   ├── style.css      # 主样式文件
│   ├── animations.css # 动画效果
│   └── responsive.css # 响应式布局
├── js/
│   ├── main.js        # 主JavaScript文件
│   ├── carousel.js    # 轮播图功能
│   └── form.js        # 表单处理
├── images/            # 图片资源
│   ├── banner.jpg     # 主视觉图
│   ├── gallery/       # 相册图片
│   └── icons/         # 图标
└── videos/            # 视频资源
    └── memories.mp4   # 纪念视频
